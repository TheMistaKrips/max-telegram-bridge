![Node.js](https://img.shields.io/badge/node->=16.0.0-brightgreen.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 🛠 Установка

### 1. Получение токенов для ботов

**Для Telegram:**
1. Найдите в Telegram бота [@BotFather](https://t.me/botfather).
2. Отправьте команду `/newbot`.
3. Введите имя бота (например, `MAX Telegram Bridge`).
4. Введите username бота (обязательно должен заканчиваться на `bot`).
5. Скопируйте полученный токен.

**Для MAX:**
Токен предоставляется преподавателем.

### 2. Установка Node.js и npm

Скачайте и установите **Node.js** с [официального сайта](https://nodejs.org/) (требуется версия 16 или выше).

Проверьте корректность установки в терминале:
```bash
node --version
npm --version
```

### 3. Создание проекта и установка зависимостей

Выполните следующие команды в терминале:

```bash
# Создаем папку проекта и переходим в нее
mkdir max-telegram-bridge
cd max-telegram-bridge

# Инициализируем проект (создаем package.json)
npm init -y

# Устанавливаем необходимые зависимости
npm install @maxhub/max-bot-api node-telegram-bot-api axios form-data mime-types
```

### 4. Создание файлов проекта

Создайте следующие файлы в корне папки `max-telegram-bridge`:
* `index.js` — точка входа
* `config.js` — конфигурация
* `queue.js` — менеджер очереди
* `max-bot.js` — MAX бот
* `telegram-bot.js` — Telegram бот
* `bridge.js` — основной мост

### 5. Настройка конфигурации

Откройте файл `config.js` и вставьте свои токены:

```javascript
module.exports = {
  // MAX бот
  max: {
    token: 'ВАШ_ТОКЕН_ОТ_ПРЕПОДАВАТЕЛЯ', 
    groupId: null
  },

  // Telegram бот
  telegram: {
    token: 'ВАШ_ТОКЕН_ОТ_BOTFATHER', 
    groupId: null
  }
};
```

## 🚀 Запуск и первоначальная настройка

Запустите проект командой:
```bash
npm start
```

**При первом запуске необходимо привязать группы:**

1. **В MAX:**
   * Добавьте бота в нужную группу.
   * Отправьте в группе команду `/setgroup`. ID группы сохранится автоматически.
2. **В Telegram:**
   * Добавьте бота в нужную группу.
   * Отправьте в группе команду `/setgroup`. ID группы сохранится автоматически.

После этой настройки двусторонняя пересылка сообщений начнёт работать автоматически.

## ⚙️ Настройки скрипта

Дополнительные параметры находятся в файле `config.js`:

| Параметр | По умолчанию | Описание |
| :--- | :--- | :--- |
| `max.token` | — | Токен MAX бота |
| `telegram.token` | — | Токен Telegram бота |
| `bridge.maxMessageLength` | `4096` | Максимальная длина сообщения |
| `bridge.queue.interval` | `1000` | Интервал отправки из очереди (в мс) |
| `bridge.queue.maxPerInterval` | `20` | Макс. количество сообщений за один интервал |
